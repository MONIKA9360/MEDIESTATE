# Property Images

## How to Add Images:

1. Save your property images to this folder
2. Name them according to the property:
   - `hospital1.jpg` - Multi-Specialty Hospital (Chennai)
   - `clinic1.jpg` - Premium Clinic Space (Bangalore)
   - `diagnostic1.jpg` - Diagnostic Center (Mumbai)

## Current Setup:

The app is configured to display images from `/images/` folder.

### To add the hospital image you shared:
1. Save the modern hospital building image
2. Rename it to: `hospital1.jpg`
3. Place it in: `public/images/hospital1.jpg`

The image will automatically appear on:
- Property cards on homepage
- Property listing page
- Property detail page

## Image Requirements:
- Format: JPG, PNG, or WebP
- Recommended size: 1200x800px
- Optimized for web (under 500KB)
